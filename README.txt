You will need to create a fiel called "apikeys.xml" in res/values to hold keys for the services you use.

E.g.:

<?xml version="1.0" encoding="utf-8"?>
<resources>
    <string name="GoogleMapsKey" translatable="false">[google API key]</string>
    <string name="WebServiceKey" translatable="false">[Web Service key]</string>
</resources>
